# 용도
======================
```
centos7 + nginx + nvm + nodejs lts + mariadb 5.x.x(utf8) + pm2 + yarn + htop 
```

# 설치
======================
```
> git clone https://github.com/neoty/vagrant-centos7-nodejs
> vagrant plugin install vagrant-vbguest
> vagrant up
> vagrant ssh
```
